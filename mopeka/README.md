
# Installation

## dependencies
```pip install --upgrade mopeka_pro_check```

## install

```make install```

And adjust settings in /etc/default/mopeka2mqtt

## get the device MAC

press Sync button and call `mopeka.py -D`
The MAC shout start with `f0:87:60:`
